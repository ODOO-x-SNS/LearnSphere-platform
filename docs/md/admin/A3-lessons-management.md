# A3 – Lessons / Content Management

## Location
Course Form → Content Tab

## UI Elements
- Lessons list (ordered)
- Lesson item shows:
  - Title
  - Type (Video / Document / Image / Quiz)
  - 3-dot menu (Edit / Delete)
- Add Content button

## Delete Flow
- Confirmation modal required

## Drag & Drop
- Lessons are reorderable
- Order affects learner flow

## States
- Empty content
- Loading
