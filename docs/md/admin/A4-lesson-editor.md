# A4 – Lesson / Content Editor

## Type
Modal Popup

## Tabs
1. Content
2. Description
3. Additional Attachments

## Content Tab
Common:
- Lesson title (required)
- Lesson type selector
- Responsible (optional)

Type-Specific:
- Video → URL + duration
- Document → File upload + Allow Download toggle
- Image → Image upload + Allow Download toggle

## Description Tab
- Rich text editor
- Displayed to learners

## Attachments Tab
- File upload OR
- External link

## Rules
- Attachments visible under lesson on learner side
- Downloads controlled by toggle

## States
- Create
- Edit
