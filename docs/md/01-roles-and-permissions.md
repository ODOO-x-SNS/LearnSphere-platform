# Roles & Permissions

## Roles
- Admin
- Instructor / Course Manager
- Learner
- Guest

## Permissions Matrix

| Feature                          | Admin | Instructor | Learner | Guest |
|----------------------------------|-------|------------|---------|-------|
| Create/Edit Courses              | ✅    | ✅         | ❌      | ❌    |
| Publish Courses                  | ✅    | ✅         | ❌      | ❌    |
| Add Lessons & Quizzes            | ✅    | ✅         | ❌      | ❌    |
| View Reporting                   | ✅    | ✅         | ❌      | ❌    |
| View Published Courses           | ✅    | ✅         | ✅      | ⚠️*   |
| Start Learning                   | ❌    | ❌         | ✅      | ❌    |
| Attempt Quiz                     | ❌    | ❌         | ✅      | ❌    |
| Add Review                       | ❌    | ❌         | ✅      | ❌    |

\* Guest visibility depends on course visibility rules.
